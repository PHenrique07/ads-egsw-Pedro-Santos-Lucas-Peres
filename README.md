# Sistema de Agenda em Python


O Sistema de Agenda é um programa simples em Python desenvolvido para gerenciar compromissos organizados por data. Ele permite adicionar, visualizar, remover e contar compromissos, garantindo que as datas inseridas sejam válidas e formatadas corretamente.

O sistema é interativo via terminal e foi construído como parte de um exercício prático de Gerência de Configuração de Software (GCS), aplicando conceitos de versionamento, colaboração com Git e GitHub e controle de mudanças.

Funcionalidades principais:

Adicionar compromisso: registra compromissos associados a uma data específica.

Visualizar agenda: exibe todos os compromissos cadastrados por data.

Remover compromisso: permite excluir um compromisso específico de uma data.

Contar compromissos: exibe o total de compromissos cadastrados.

Validação de data: garante que a entrada siga o formato dd/mm/aaaa e seja uma data real.

Tecnologias utilizadas:

Python 3

Git e GitHub

Git Bash / GitHub Desktop
